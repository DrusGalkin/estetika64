# Салон красоты
## Стек технологий:
Redis, PostgreSQL, gRPC, GoLang, Axios, Docker, JWT, Fiber, SwiperJS, Tailwind
## Функционал сервера *не для продакшена*:
- Загрузка изображений
- Аутентификация
- Миграции
- Работа с пользователями (Разбито на 2 микросервиса, auth и users)
- Система комментирования записей
## Запуск
- Перейти в папку ./backend/mail-sender
  - установить пароль в .env для SMTP подключения
  - Два шага назад cd ../../
- Перейти в папку ./frontend
  - установить ключ в .env для reCaptcha
  - Снова cd ../../
- docker-compose up --build
